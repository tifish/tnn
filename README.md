# tnn：Total Commander 操作习惯的 nnn

[English](README.en.md)

感谢 Arun Prakash Jana 开发的 [nnn](https://github.com/jarun/nnn)，让我可以在远程登录 Linux 系统时继续使用 Total Commander 的操作习惯。

tnn 是 nnn 的 fork，主要目标是提供接近 Total Commander 的操作习惯，并尽量减少对 nnn 的修改。

## 修改内容

- 始终保持在过滤模式下。
- 没有过滤字符时，`BACKSPACE` 返回上一级目录。
- `ESC` 清除过滤字符。
- `ENTER` 直接执行可执行文件。
- `F2` 重命名文件。
- `F3` 显示文件详细信息。
- `F4` 编辑文件。
- `F5` 复制选中文件到当前目录。
- `F6` 移动选中文件到当前目录。
- `DEL` 删除文件。
- `Ctrl+N` 创建新文件/链接/目录。
- `Alt+M` 批量重命名。
- `Ctrl+D` 显示书签列表。
- `Ctrl+B` 把当前目录添加到书签列表。
- `Ctrl+R` 刷新文件列表。
- `Ctrl+G` 进入 shell 命令行。
- `SPACE` 选中当前文件。
- `Ctrl+A` 选中所有文件。
- `Alt+-` 取消选择。
- `[` 访问上次标记的目录。
- `Ctrl+X` 退出程序。
- `Alt+1` .. `Alt+8` 切换 tab 页。

## 未改动的原有快捷键

- `*` 切换可执行权限。
- `'` 跳转到第一个非目录文件。
- `+` 切换文件选择状态。
- `,` 标记当前目录。
- `.` 显示隐藏文件。
- `;` 运行插件。
- `=` 启动应用程序。
- `>` 导出文件列表。
- `@` 访问起始目录。
- `]` 执行单个命令。
- `` ` `` 访问根目录。
- `~` 跳转到 HOME 目录。
- `Alt+C` 通过 SSHFS 连接远程服务器。
- `Alt+U` 卸载 SSHFS 连接。
- `TAB` 切换 tab 页。
- `Alt+D` 文件列表显示详细信息。
- `Alt+Z` 生成压缩包。
- `Alt+T` 排序文件。
- `Ctrl+O` 使用指定应用打开文件。
- `;` 运行插件。
- `Alt+S` 保存/加载会话。
